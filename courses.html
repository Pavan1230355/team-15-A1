HTML (HyperText Markup Language)

HTML is the foundation of all web pages. It structures the content using elements (tags). Every website you see is built with HTML at its core.

Core Concepts

  • HTML uses tags like <h1>, <p>, <div> to structure content.
  • Attributes like id, class, src, and href add functionality.
  • Forms allow user input with <input>, <textarea>, etc.

Example

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Welcome to HTML</h1>
  <p>This is a paragraph.</p>
</body>
</html>

Why Learn HTML?

Without HTML, no web page can exist. It is the skeleton of the web.

CSS (Cascading Style Sheets)

CSS controls the design and layout of web pages. It makes websites attractive, responsive, and user-friendly.

Core Concepts

  • Selectors (.class, #id, tag) target HTML elements.
  • Properties like color, margin, padding, font-size.
  • Responsive design using media queries (@media).

Example

body {
  background-color: lightblue;
  font-family: Arial;
}
h1 {
  color: darkblue;
}

Why Learn CSS?

CSS allows websites to look modern, professional, and adapt to different screen sizes.

JavaScript (JS)

JavaScript makes web pages interactive. It can update content dynamically, handle user inputs, and control multimedia.

Core Concepts

  • Variables (var, let, const).
  • Functions to reuse code.
  • DOM Manipulation (document.getElementById, addEventListener).

Example

<button onclick="sayHello()">Click Me</button>

<script>
function sayHello() {
  alert("Hello, JavaScript!");
}
</script>

Why Learn JavaScript?

JavaScript is the backbone of interactive web applications and is used in almost every website today.

MongoDB

MongoDB is a NoSQL database that stores data as JSON-like documents. It is highly scalable and flexible.

Core Concepts

  • Collections (like tables in SQL).
  • Documents (like rows in SQL, but stored as JSON).
  • CRUD Operations: Create, Read, Update, Delete.

Example

db.students.insertOne({ name: "Alex", age: 21, course: "JS" });
db.students.find();

Why Learn MongoDB?

MongoDB is widely used in modern apps, especially with Node.js, for building scalable applications.

MySQL

MySQL is a popular relational database management system. It uses structured tables and SQL queries.

Core Concepts

  • Tables consist of rows and columns.
  • SQL queries manage data (SELECT, INSERT, UPDATE, DELETE).
  • Joins combine data from multiple tables.

Example

CREATE TABLE students(
  id INT PRIMARY KEY,
  name VARCHAR(50),
  age INT
);

INSERT INTO students VALUES(1, 'Alex', 21);
SELECT * FROM students;

Why Learn MySQL?

MySQL powers many enterprise applications, websites, and CMS platforms like WordPress.

Java

Java is a powerful, object-oriented programming language used for building applications from mobile apps to large-scale enterprise systems.

Core Concepts

  • Object-Oriented Programming (Classes, Objects, Inheritance).
  • Platform independence (Write Once, Run Anywhere).
  • Robust standard library for networking, GUI, and more.

Example

class Hello {
  public static void main(String args[]) {
    System.out.println("Hello, Java!");
  }
}

Why Learn Java?

Java is widely used in Android apps, banking software, and enterprise solutions due to its performance and reliability.
