# Contributing to QuickQuiz

Thank you for your interest in contributing to QuickQuiz! We welcome contributions from everyone.

## ğŸš€ How to Contribute

### 1. Fork the Repository
- Click the "Fork" button at the top right of this repository
- This creates your own copy of the project

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR-USERNAME/team-15-A1.git
cd team-15-A1
```

### 3. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes
- Add new features
- Fix bugs
- Improve documentation
- Add new quiz questions
- Enhance UI/UX

### 5. Test Your Changes

**Backend:**
```bash
cd backend
npm install
npm start
```

**Frontend:**
- Open `frontend/indexx.html` in your browser
- Test all features

### 6. Commit Your Changes
```bash
git add .
git commit -m "Description of your changes"
```

### 7. Push to Your Fork
```bash
git push origin feature/your-feature-name
```

### 8. Create a Pull Request
- Go to the original repository
- Click "New Pull Request"
- Select your branch
- Describe your changes
- Submit!
